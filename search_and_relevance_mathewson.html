Unterschiede zwischen Schreiben für Print und für das Web

Entscheidung für die Lektüre

  • Print Medien: Erkennen der Relevanz vor dem Lesen

  • Web: Lesen vor dem Erkennen der Relevanz

Weg durch die Information

  • Print: Autor bestimmt Informationsweg

  • Web: Leser bestimmt Informationsweg

User-Bewegung auf einer Website

Abgeschlossenheit

  • Print: Geschlossene Texte für vollständige Lektüre

  • Web: Nutzer suchen nach Informations-Nuggets

Vorverständnis

  • Print: Autor kann die Bedeutung von Ausdrücken bestimmen

  • Web: Die Leser bestimmen die Bedeutungen

Weg zur Information

  • Print: Physischer Kontakt, Empfehlung wichtigster Weg zur Information

  • Web: Suche wichtigster Weg zur Information

Spezifische Information

  • Print: Inhalte decken meist die Bedürfnisse unterschiedlicher Nutzergruppen ab.

  • Web: Spezifische Inhalte ranken am besten.

Suche als Schlüssel für Userbedürfnisse

  • Suchmaschinenoptimierung beginnt immer mit der Festlegung der Keywords bzw. der Fragen der User.

  • Suche und Suchverhalten liefern die wichtigsten Daten über die User einer Website.

Keyword-Clouds

  • Vor dem Verfassen einzelner Inhalte sollten alle relevanten Keywords/Fragestellungen erhoben werden.
  • Es dürfen sich nicht unterschiedliche Inhalte zum selben Keyword Konkurrenz machen.

Optimierung auf einzelne Keywords

  • User und Suchmaschine müssen erkennen, zu welcher Abfrage ein Inhalt am besten passt.
  • Am höchsten bewertet wird der Inhalt, der eine bestimmten Suchabsicht am genauesten erfüllt.

Schreiben als Weg zum Publikum

Ziel: Engagement

  • Das Ziel besteht nicht im bloßen Volumen, sondern in Interaktionen.
  • Wenn die Inhalte erstellt werden, muss klar sein, welche Interaktionen der User erwartet werden.

Fortdauernde Beziehung zwischen Autor und Schreiber

  • Online-Texte sind Live-Texte.
  • Durch das Feedback der User kann sich der Text laufend verändern.

Relevanz

Verhältnis zum Kontext

  • Relevanz entsteht durch das Verhältnis einer Information zu einem Kontext.
  • Für User machen die relevantesten Informationen die wenigste Mühe.
  • Relevante Information heben sich von ihrem Kontext deutlich ab.

Suchmaschinen als Weg zur Information

  • Die meisten User finden Informationen nicht über Startseiten, sondern über Suchergebnisse.
  • Titel, URL und Snippet müssen zeigen, was der User findet
  • Der Inhalt muss die Erwartungen der User sofort erfüllen.

Intentionen und Inhalt

  • User suchen immer mit einer bestimmten Absicht.
  • Sie müssen sofort erkennen, ob eine Seite ihrer Absicht entspricht.
  • Optimal ist, wenn die SEO-relevanten Elemente Keywords und Intention verbinden.

Links und Navigation

Verteilung der Informationen

  • Innerhalb der für einen Auftritt gewählten Keywords sollen sich die User vom Allgemeinen zum Besonderen bewegen.
  • Jeder Inhalt sollte relevante Links zu den anderen Inhalten eines Angebots enthalten.
  • Die Linktexte beschreiben für User und Google die anderen Inhalte - sie müssen sprechend und klar sein.

Externe Verlinkung

  • Eingehende und ausgehende Links gehören zum Kontext einer Information.
  • Eingehende Links (Backlinks) sind wesentlich für die Glaubwürdigkeit eines Auftritts.

Sprache

Autorensprache vs. Lesersprache

  • Es werden nur Inhalte in der Sprache der User gefunden.
  • Organisationsinterne Bezeichnungen sind ein Haupthindernis für viele User.

Quellen für usergerechte Sprache

  • User suchen in der Sprache, die sie in sozialen Medien verwenden.
  • Die internen Suchen sollten permanent ausgewertet werden.

Optimierung als Daueraufgabe

Veränderung von Abfragen

  • Suchabfragen sind Trends unterworfen.
  • Die Suche ist von Änderungen in der Terminologie, Markenbezeichnungen etc. besonders betroffen.

Lernen aus der Interaktion

  • Userbedürfnisse lassen sich nur empirisch erheben.
  • Die ersten Versionen von Inhalten sind nur Ausgangspunkte für Interaktionen.
